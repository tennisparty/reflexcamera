Battery

The system runs on a 12v battery which must be bought seperately. The system can run on any battery chemistry which gives great flexibility, a very basic chemistry comparison is provided below:

Lithium-ion Lead-acid AGM Gel

The system run time is dependent on the capacity (ampHours) of the battery and camera settings. The number of nights it is possible to record will be dependent on the battery temperature, night length and settings. Higher battery capacities mean the battery will weigh more. During testing an 11kg battery seems to be a good compromise between allowing navigation of a woodland with the battery in a backpack and a good system run time.
